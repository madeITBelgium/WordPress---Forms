Build cool, easy and flexible forms with Forms.

== Description ==

Forms is an easy form manager that lets you manage all your cool forms. It's easy to create your own contact form or newsletter subscriber.

== Installation ==

1. Upload the entire `madeit-forms` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Forms' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](https://www.madeit.be/producten/wordpress/forms-plugin/).
